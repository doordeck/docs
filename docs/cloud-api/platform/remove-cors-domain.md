# Remove CORS Domain

:::warning
This endpoint is only available to users with Doordeck issued auth tokens.
:::

Remove CORS domain.

### HTTP Request

`DELETE https://api.doordeck.com/platform/application/APPLICATION_ID/cors`

### Request Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| url       | true     | CORS Domain |
